The term project involved analysing transcriptomic data pertanining to 16 cancer types of interest from the TCGA Pan-Cancer Atlas.
<br>
</br>
The techniques implemented for this analysis included Dimensionality Reduction (for Visualisation), Differential Gene Expression Analysis,
Pathway Enrichment Analysis, and Gaussian Naive-Bayes Classification.
<br>
</br>
